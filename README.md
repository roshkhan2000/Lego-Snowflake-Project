# Lego Snowflake Project 
A data engineering oriented project that loads data in to Snowflake for analysis. 
<br>

### :a: About

Steps carried out:
- Created Stages
- Created Tables
- Loaded data from Stages to Tables
- Added PKs and FKs
- Created an ER diagram using DBeaver
- Custom View for Data Analysis
